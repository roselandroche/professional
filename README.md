## Personal Project: Professional History

The goal of this project is to create a safe, convenient, and central place for people to save the information they need when searching for a job.

This information includes, but is not limited to, job history and the details thereof, references and their contact information, education history and the details thereof, volunteer history. A stretch goal of this project is to construct an extremely well protected space to save tax documentation, for ease of tax reporting and audits.

# Deployed at:
